    2.1、安装git

    在终端中输入git --version来查看你的电脑上是否有git环境，如果出现了如下提示，则证明本地有git环境，跳过第1步，直接安装Node.js

    安装git有两种方法

    • 官网下载安装包git官网
    • 用homebrew安装(homebre的安装教程官网上写的一清二楚)，命令为brew install git
